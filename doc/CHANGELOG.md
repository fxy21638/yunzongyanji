# 更新日志 (CHANGELOG)

本文档记录 KEYU STC32G144K246 开源程序库的所有重要更新和变更历史

---

## [2.1.0] - 2026-04-23

### 新功能 (Added)

- **新增应用示例**
  - `E14_cdc_send_images_to_vofa` - USB CDC 发送图像数据到 VOFA+ 上位机示例
  - `E15_icm45686_get_angle` - ICM45686 陀螺仪三轴角度积分示例

### 改进 (Changed)

- **系统时钟配置优化**
  - `config.h` 中 `FOSC` 宏改为引用 `system_clock_freq` 变量，提高时钟配置灵活性

---

## [2.0.10] - 2026-04-08

### 修复 (Fixed)

- 修复舵机例程 ANGLE_TO_DUTY 宏计算结果错误的问题，将浮点运算改为整数运算
- 修复软件 I2C 初始化时 delay_us 断言误触发的问题

---

## [2.0.9] - 2026-03-28

### 新功能 (Added)

- **无线串口库增强**
  - 添加 RTS (Request to Send) 硬件流控支持
  - 支持大于 255 字节的数据传输 (参数类型改为 uint16_t)
  - 在发送数据时自动控制 RTS 信号

### 改进 (Changed)

- **例程文件夹重命名**
  - 统一例程命名规范，使用 E01_、E02_ 前缀
  - E01_adc - ADC 模数转换示例
  - E02_btn7971b - BTN7971B 电机驱动示例
  - E03_encoder - 编码器示例
  - E04_icm42686 - ICM42686 六轴 IMU 示例
  - E05_icm45686 - ICM45686 六轴 IMU 示例
  - E06_ips200_display - IPS LCD 屏幕显示示例
  - E07_key - 按键示例
  - E08_pit - 定时器中断示例
  - E09_servo - 舵机示例
  - E10_usb_cdc - USB 虚拟串口示例
  - E11_wireless_com - 无线串口透传示例
  - E12_mt9v034_ips200_display - MT9V034 摄像头 IPS 显示示例
  - E13_mt9v034_process_images - MT9V034 图像处理示例

- **文件编码统一**
  - 统一所有源代码文件为 GB2312 编码
  - 修复中文注释在 Keil C251 环境下显示乱码的问题

### 修复 (Fixed)

- 修复无线串口发送长度限制问题（从 uint8_t 改为 uint16_t）
- 修复 RTS 引脚初始化问题
- 优化文件头注释格式

---

## [2.0.8] - 2026-02-04

### 新功能 (Added)

- **新增 IMU 传感器支持**
  - 添加 `ky_icm42686` 驱动支持 TDK ICM-42686 六轴惯性测量单元
  - 添加 `e_icm42686` 应用示例

### 改进 (Changed)

- **SPI DMA 传输优化**

  - 优化 DMA 传输配置效率
  - 加快大量数据传输时的中断处理

## [2.0.7] - 2026-01-28

### 新功能 (Added)

- **新增 IMU 传感器支持**
  - 添加 `ky_icm42688` 驱动支持 TDK ICM-42688 六轴惯性测量单元

- **新增软件模拟 I2C 驱动 (`ky_soft_i2c`)**
  - 支持 GPIO 模拟 I2C 时序，不占用硬件 I2C 模块
  - 可自定义 SCL 和 SDA 引脚
  - 遵循标准 I2C 时序，支持 100kHz/400kHz 速率

- **IMU 原始数据获取功能**
  - 传感器驱动支持直接获取原始数据
  - 添加原始加速度和角速度获取接口
  - 方便用户进行自定义数据处理和校准

### 改进 (Changed)

- **显示驱动优化 (`ky_ips`)**
  - 优化 `ips_clear()` 函数，使用 DMA 全屏刷新方式
  - 预分配切换缓冲区空间，减少屏幕切换速度
  - 加快后续每次屏幕显示效果

- **矩形绘制功能增强**
  - 优化 `ips_fill_rectangle()` 函数，新增 `enable` 参数
  - 支持实心/空心矩形模式：
  - `enable=0` 实心矩形，`enable>0` 空心矩形，值为边框宽度
  - 自动处理边界裁剪，防止溢出屏幕

- **摄像头驱动优化 (`ky_mt9v034`)**
  - 优化摄像头帧缓冲配置，减少采集时间
  - 优化图像显示函数 `ips_show_gray_image()`，启用 DMA 加速传输
  - 改进图像格式转换算法，减少内存拷贝开销
  - 提升整体图像刷新效果
  - 新增可配置参数接口，方便根据场景优化配置

- **清理脚本优化**
  - 修改 `clean.bat` 清理脚本路径配置
  - 增加对更新后路径的处理支持
  - 简化脚本逻辑，提高容错性


### 修复 (Fixed)

- 修复清理脚本在特定路径下执行出错的问题
- 修复文件头部注释在 Windows CMD 下的乱码问题
- 修改构建目录名称，将 `.bulid` 改为 `bulid`

## [2.0.0] - 2026-01-05

### 重大变更 (Breaking Changes)

- 全新重构架构，所有模块添加 `ky_` 前缀，遵循命名规范
- 重构项目目录结构，采用分层架构 (`core` / `drivers` / `devices` / `common`)

### 新功能 (Added)

- **驱动层 (drivers)**
  - `ky_gpio` - GPIO 驱动，支持推挽/开漏/上下拉配置
  - `ky_uart` - UART 驱动，支持 DMA 收发
  - `ky_spi` - SPI 主模式驱动，支持 DMA 传输
  - `ky_pwm` - PWM 输出驱动
  - `ky_adc` - ADC 模数转换驱动
  - `ky_timer` - 定时器驱动
  - `ky_encoder` - 编码器接口驱动
  - `ky_delay` - 精确延时驱动
  - `ky_i2c` - I2C 通信接口
  - `ky_system_clock` - 系统时钟驱动
  - `ky_usb_cdc` - USB 虚拟串口

- **设备层 (devices)**
  - `ky_ips` - IPS LCD 显示驱动，支持文字和图形显示
  - `ky_mt9v034` - MT9V034 灰度摄像头，支持 FIFO 模式和 DMA 采集
  - `ky_icm45686` - ICM45686 六轴 IMU 驱动
  - `ky_lsm6dsr` - LSM6DSR 六轴 IMU 驱动
  - `ky_imu_base` - IMU 通用接口层

- **应用示例 (app_examples)**
  - LED 闪烁示例
  - IPS 屏幕显示示例
  - MT9V034 摄像头采集示例
  - USB CDC 虚拟串口示例

### 改进 (Changed)

- 统一命名规范
- 标准化文件头注释格式
- 优化头文件包含顺序，减少编译依赖
- 改进 GPIO 驱动引脚命名规则
- 优化 UART DMA 收发实现

### 修复 (Fixed)

- 修复 IPS 屏幕绘图算法错误，避免显示假影
- 修复 MT9V034 图像位置偏移问题，修正像素偏移
- 修复 MT9V034 DMA 采集在 Large 模式下的内存地址计算
- 修复 I2C 通信协议实现中的时序问题
- 修复 FIFO 读写指针复位后写入错误
